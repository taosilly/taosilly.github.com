互联网学习的10个高质量网站

发表于 2016-11-27 | 更新于 2016-11-27 | 分类于 思想 | | 阅读次数 | 阅读次数

时间过的太快了，只要你没利用好，一回头…好宝贵的时间呀！想起来都是憋在心里无法流出的泪。

想想自己，从初中、高中、大学、到现在，有多少的时间被浪费了，没有换来金线，也没有换来经验，更没有换来好的生活。再看到别人，我有时候在想，是不是我的心智比正常慢了几年呢！

身在现在的互联网中，你不得不紧跟步伐，不管是人际、文化、技术、生活、思维等。

我是一个做编程出生的人，所以下面就我了解的比较好的学习网站分享一下。

1. 简书 http://www.jianshu.com

使用最频繁的就是简书了，我每天都想打开简书看几篇文章，也喜欢在简书写写文章。在简书 可以关注喜欢的专题、作者，来让显示的文章都变成自己喜欢的。上面有各种各行业的好文章，加上 评论、打赏、收藏、简信等 强大的功能，总之就是既好玩又实用。

2. 知乎 https://www.zhihu.com/

知乎是一个真实的网络问答社区，社区氛围友好与理性，连接各行各业的精英。用分享着彼此的专业知识、经验和见解，为中文互联网源源不断地提供高质量的信息。

3. 慕课网http://www.imooc.com/

慕课网是垂直的互联网IT技能免费学习网站。以独家视频教程、在线编程工具、学习计划、问答社区为核心特色。在这里，你可以找到最好的互联网技术牛人，也可以通过免费的在线公开视频课程学习国内领先的互联网IT技术。

4. 麦子学院http://www.maizitime.com/

线上教育机构，同时也提供了大量的视频教程，质量也挺高的。

麦子学院，国内第一家在美国建立商务中心的IT在线教育机构，目前已与美国知名教育公司取得合作，未来将源源不断向国内输出大量高质量教育资源 前身麦可网，2014年，麦可网完成千万级A轮融资，并更名为”麦子学院“，同时通过“麦子圈”IT职业实名社交圈——提供包括企业招聘对接，猎头，项目外包，经验分享，职业交友等一系列增值服务。 做在线职业教育示范学院，将教育和课程做到极致。除了提供高质量的课程学习之外，也提供包括就业推荐，职业交友，项目外包，创业服务等全面的增值服务，并且线上线下结合，移动设备和传统网络相结合，打造IT职业教育的一个完整生态链。 以高端IT技术型人才培养及服务为核心，探索及倡导技术交流创新模式。提供的不仅仅是技能培养，而是务实的职业导向。

5. 网易云课堂http://study.163.com/

网易云课堂，是网易公司打造的在线实用技能学习平台，该平台于2012年12月底正式上线，主要为学习者提供海量、优质的课程，用户可以根据自身的学习程度，自主安排学习进度。

6. 极客学院http://www.jikexueyuan.com/

极客学院是中国Android开发在线学习平台，汇集了几十名国内顶尖的有多年项目和实战经验的Android开发授课大师，精心制作了上千个高质量视频教程，涵盖了Android开发学习的基础入门、中级进阶，高级提升、项目实战开发等专业的android开发课程。

7.中国大学MOOChttp://www.icourse163.org/

中国大学MOOC是由网易与高教社携手推出的在线教育平台，承接教育部国家精品开放课程任务，向大众提供中国知名高校的MOOC课程。在这里，每一个有意愿提升自己的人都可以免费获得更优质的高等教育。

8. 推酷http://www.tuicool.com/

推酷网是一款个性化阅读网站，目标用户是IT领域人群，能够自动聚合和分析来自新闻站点、博客、微博的内容，并通过对用户的阅读兴趣进行智能分析，向用户个性化推荐科技资讯、技术文章、产品设计、营销推广等内容。用户通过注册并绑定微博，选择感兴趣的文章主题，网站算法自动推荐合适的文章。用户可对内容进行“喜欢”、收藏、转发等操作，算法根据用户的操作不断调整和优化推荐条目，为用户提供个性化的阅读

9. 春秋 http://www.ichunqiu.com/

i春秋网络安全培训学院是中国最有影响力的网络安全培训机构。专业提供网络安全培训、信息安全培训、网络安全工程师培训。i春秋，培育信息时代的安全感！

10. segmentfault https://segmentfault.com/

SegmentFault ( www.sf.gg ) 是中国领先的开发者技术社区。一个纯粹、高质的技术交流的平台。

另外
腾讯视频 百度传课， 个人感觉不是很好，内容太杂，质量也不高。

说了这么多，其他我平常也没有全用到。 简书，慕课网，麦子，网易云课堂，极课学院，知乎多多少少经常接触，其他没怎么用过。不过，个人感觉如果感兴趣的话还是可以去看看的。等待你的发掘吧…
